#Zef Style
### An experiment in importing React components from npm, via webpack, to a Clojurescript/Reagent app. 

webpack build: 

```bash
npm install && npm run build
```

leiningen build (using fighweel):

```bash
lein fighweel
```

while it's running, open `public/index.html`
